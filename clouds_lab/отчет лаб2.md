Как и с AWS, сначала узнаем что такое Azure - это публичное облако от Microsoft, предоставляющее более 
200 сервисов для вычислений, хранения, аналитики, ИИ, сетей и безопасности

Ключевые категории сервисов:
- Compute (Вычисления): VM (A/D/F/G/H/N-series), Batch, App Service, Functions
- Storage (Хранение): Blob, Files, Disks, Data Lake
- Analytics (Аналитика): Databricks, Data Factory, Stream Analytics, HDInsight
- Databases (Базы данных): PostgreSQL, MySQL, Cosmos DB, SQL Database
- Networking (Сети): CDN, Load Balancer, VPN Gateway
- Monitoring (Мониторинг): Azure Monitor, Application Insights
- Backup (Резервное копирование): Recovery Services Vault

Примеры идентификации сервисов:

```
Microsoft.DBforPostgreSQL → Azure Database for PostgreSQL
Microsoft.DataFactory + "Data Factory" → Azure Data Factory  
Microsoft.Cache + "Redis Cache" → Azure Cache for Redis
```
```
Meter Category: Backup → https://azure.microsoft.com/pricing/details/backup/
Meter Name: "GRS Data Stored" → GRS Backup Storage GB
```
```
IT-Tower: Analytics (Data Factory/Databricks/Stream Analytics)
IT-Tower: Compute (VM A/D/F/G/H/N-series, Batch)
IT-Tower: Backup (RecoveryServices Vault)
```
